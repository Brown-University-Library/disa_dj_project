Welcome to Stolen Relations

This project, housed at Brown University, is a collaborative effort to build a database record of enslaved indigenous people throughout time, all across the Americas. Thank you for contributing to this project.

Here's how to start adding information:

  • If you are entering information from a document that has not previously been added: you are creating a new source.
  • if you are editing a document that you or someone else previously created: search under Open previous .
  • To list the documents that anyone entering information into Stolen Relations has created: toggle Show only your entries off.

For more information on using this form, open the style guide in a new tab . You can report problems or ask for help in a new tab.
